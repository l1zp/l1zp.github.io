final数据

  • 对于基本类型，final使数值恒定不变
  • 对于对象引用，final一旦引用被初始化一个对象，就无法改变指向另一个对象。到那时对象其本身的值是可变的
关于static和final
  • static 主要是唯一性，final则是不可变
